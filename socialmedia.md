## LinkedIn

Just shipped a **World Bank Countries Explorer** built with Streamlit! 🌍

It started as a simple data fetch-and-display app — but I wanted it to look as good as it works. Here's what I added in the latest update:

- **Sidebar filters** for Region and Income Level
- **4 metric cards** showing live counts (countries, regions, income levels, capitals)
- **Interactive Plotly charts** — a horizontal bar chart and a donut chart, both updating as you filter
- **Custom theme** with a clean blue palette and styled metric cards
- All powered by the free **World Bank REST API** (no auth needed)

Stack: Python · Streamlit · Plotly · Pandas · Poetry

The whole thing is open source — drop a comment if you'd like the repo link!
